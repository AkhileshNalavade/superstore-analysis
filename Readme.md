# Superstore Sales & Profit Analysis

## Objective
Analyze superstore sales data to understand monthly trends, regional performance, and profitability using Python.

## Dataset
The dataset contains order-level data with the following fields:
- order_date
- region
- category
- sub_category
- sales
- profit

## Analysis Performed
- Converted order_date to datetime for time-based analysis
- Created monthly sales and profit trends
- Analyzed sales vs profit across regions
- Identified loss-making orders
- Evaluated category-wise profitability
- Visualized trends using line and bar charts

## Key Insights
- Sales growth does not always result in profit growth
- Technology category contributes the highest profit
- Some regions generate revenue but require cost optimization
- Loss-making orders indicate pricing or discount issues
- Monthly trends help understand business seasonality

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
